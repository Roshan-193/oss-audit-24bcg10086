## Description of Scripts  

### Script 1: System Information Viewer  
Displays system details such as user name, operating system, kernel version, uptime, and current date and time.

### Script 2: Git Installation Checker  
Detects whether the user has Git that is enabled or not and displays its version along with the short message.

### Script 3: Storage and Permission Checker  
It analyzes directories to show space use and permissions and checks for the existence of git configuration files.

### Script 4: Log Analyzer Tool  
The program searches through a log file to find a specific keyword which it then uses to calculate how many times the keyword appears while showing the last five lines that match the search term.

### Script 5: Open Source Statement Generator  
This is a simple program that lets a user in writing down lots of open-source statements saved in a text file.
